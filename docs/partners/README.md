# Sandi For Partners

These docs are for people who interact with Sandi as a household companion,
collaborator, or shared-space participant. They describe her outward behavior:
how conversations work, how memory feels, what commands mean, and how a live
Sandi can become specific to the people around her.

Start here:

- [Interacting with Sandi](interacting.md): Discord conversations, commands,
  memory, reminders, todo lists, and shared-space expectations.
- [Personalizing Sandi](personalization.md): how a deployment can shape Sandi's
  soul, boundaries, memory, reusable skills, and local habits.

For installation, deployment, source changes, and runtime internals, see
[Sandi for developers](../developers/README.md).
